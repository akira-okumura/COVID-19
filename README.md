# 新型コロナウイルスの感染経路図置き場

このレポジトリでは、Twitter で公開している新型コロナウイルスの感染経路を可視化した図の PDF ファイルを公開しています。

## 名古屋大学基金へのご寄付のお願い

名古屋大学では、コロナ禍での経済困窮学生を支援するため「新型コロナウイルス感染症対策緊急学生支援基金」を開設しています。
